Overview

I chose this specific work sample because first and foremost, I'm proud of the way the application turned out, a fully playable web interface for a game I played a lot when I was a kid. Secondly, I selected this project because of the unique challenges it presented that we had to overcome as a group

Challenges I (We) Faced

As I previously said, there were some unique challenges presented for this project. First off was the challenge of creating a solid team environment and making sure we all knew our parts and could communicate efficiently as a team. Another challenge was using source control effectively to maintain a code base for an application that 3 people were coding simultaneously.

Lastly, this project made us think critically about documentation and how we were to handle this project, being split over 2-3 week long "Sprints". We had to effectively break down the project in to individual components to clearly outline every deliverable for the project

What I Learned

This project really taught me the value and importance of using source control effectively, and furthermore made me more proficient with using these tools. We, as a group, also learned a hard lesson in project planning and scope, as well as time management. The struggle we had with meeting hard deadlines and dealing with unforeseeable issues like emergency situations, clearly showed the great importance of managing a project efficiently from Day 1.
