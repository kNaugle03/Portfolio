Overview

This work sample is the first assignment I got to tackle in our Web Application Development course. The purpose of this assignment was to create a front end interface with a MySQL backend via PHP. I chose this sample because I believe it is an excellent demonstration of all of the skills I had gained up until that point in the course.

Challenges I Faced

One of the largest challenges I had to deal with during this assignment was trying to figure out how to persist data between pages for pagination, sorting, and editing individual records. The introduction of a data layer added a level of complexity to coding that I hadn't come up against before, and I really quite enjoyed the challenge.

What I Learned

While completing this assignment, one of the things I was most proud of was my solution for pagination and sorting records. It took a long time to figure out how to maintain the state of the sorted records along with the page numbers, but finally getting it to work 100% was incredibly satisfying. I also realized that I quite like working with data and solving the issue of displaying it effectively.
